# Contains Duplicates
Write a function called `containsDuplicates` that accepts an array and returns `true` or
`false` depending on if the array contains any duplicate values.

## Sample IO
```
input: ["hippo", "cat", "dog", "cat"]
output: true

input: [1, 2, 3, 4, 5, 2]
output: false

input: [1, 2, 3, 4, 5]
output: false

input: []
output: false
```
