# Balanced Brackets

Generate all balanced bracket combinations
print all possible balanced parenthesis combinations up to a number

Example IO:
```
balancedBracket(2);
=> (()), ()()
balancedBracket(3);
=> ((())), (()()), (())(), ()(()), ()()()
```
